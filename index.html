Projects

Price Imputation Model

As part of a data science team, I partnered with a humanitarian aid organization to develop a robust imputation model. The model uses granular cost-of-living data in war-afflicted regions to predict the price of basic survival goods. This in turn helps inform more effective delivery of cash-based assistance during conflicts.

Tools Used: Python, R, Tableau, Pandas, Linear Regression, k-Nearest Neighbors

Gaussian Process Regression in Chemistry

I implemented a Gaussian Process Regression routine to interpolate and predict molecular properties of chemical species. I also sped up the acquisition of accurate data by engineering code that connects this regression technique to other commonly-used chemistry software packages.

Tools Used: Python, Git, Matplotlib, Kernel Regression, OpenMPI, Computer Cluster

Recommendation System for Fantasy Soccer

This project involved scraping professional soccer player data from an API and analyzing both team and player performance. The relevant data was then aggregated and visualized in order to recommend the top players for use in a fantasy sports league.

Tools Used: Jupyter, Pandas, Seaborn

Sustainable Filters for Drinking Water

I initiated the formation of an international team that provided drinkable water for six Nepalese households using state-of-the-art water filters. I also organized the project into relevant work packages and interviewed candidates in the Kathmandu valley.

Tools Used: Agile Project Management, Gantt Chart, International Collaboration
